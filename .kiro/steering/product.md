# Product Overview

NFT Gift Protocol is a decentralized gifting system that replaces traditional gift cards with customizable, AI-generated NFT art that holds transferable ERC-20 tokens.

## Core Concept

Users create gift card NFTs with AI-generated artwork and embedded token vaults. Recipients can hold the NFT as collectible art or liquidate it to extract the token value.

## Key Features

- AI-generated gift card artwork using Stable Diffusion
- ERC-721 NFTs with embedded ERC-20 token vaults
- Email-based gift card delivery and claiming
- Marketplace for artist designs and secondary NFT sales
- Liquidation mechanism to extract token value
- Decentralized storage using IPFS

## Target Network

**Polygon Amoy Testnet Only** (Chain ID: 80002)
- This is a testnet-only
- No real tokens or value are involved
- All transactions use test MATIC

## User Flows

1. **Create Gift**: Generate AI art → Mint NFT → Embed tokens → Send via email
2. **Receive Gift**: Claim via email → View in wallet → Hold or liquidate
3. **Marketplace**: Browse artist designs → Purchase templates → Trade NFTs
4. **Liquidate**: Extract token value from owned gift cards
